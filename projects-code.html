Coding Projects

All of these projects are online, so go check them out and play them at My Github

VoogaSalad

A game building and running environment, primarily for grid-based games such as Pokemon. The editor is extremely versatile and allows for different types of tiles to be placed in the space. There are options for NPC's with spoken text, different obstacles, decorations, teleportation linkages, and switches which open doors.

SLogo

A simple Logo command interpreter. The onscreen turtle will move according to the commands it is given. The commands range from the simplest "forward" and "turn" commands to much more complicated ones, such as for loops, creating variables, implementing multiple turtles at once, and deselecting and selecting turtles by clicking on them.

Mega Invaders

My first game made using JavaFX. This Space Invaders-inspired game takes in direct keyboard input and uses it to control your spaceship to move around and fire missiles at the approaching enemies. All enemies and missiles are objects, and collisions are detected between objects to decide whether the missiles hit their target.
